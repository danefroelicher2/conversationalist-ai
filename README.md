# Conversationalist AI

Intelligent security system with voice recognition and computer vision.

## Setup Complete
- Python 3.12.3
- PostgreSQL 16
- Database: conversationalist_ai

## Quick Start
```bash
# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Setup database
python scripts/setup_database.py

# Test connection
python scripts/test_connection.py
```
